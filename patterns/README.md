# Loop Patterns

Patterns describe repeatable agent loops: what triggers the loop, which outputs it produces, who reviews it, and which guardrails stop it from becoming uncontrolled automation.

Built-in Python patterns are available with:

```bash
loop patterns
loop patterns pr-babysitter
```

Start with report-only or reviewed modes, then graduate only after several successful runs with clear evidence in `loop-run-log.md`.
